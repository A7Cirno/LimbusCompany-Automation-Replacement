# -自动将“黎明将至”替换为“天将拂晓”
下载“使用到的程序”


将程序放到游戏文件的Lang目录下面


然后生成快捷方式


Win+R打开运行栏，输入shell:startup，将快捷方式放到启动文件夹下面，重新装一遍汉化即可生效


之后不论重装还是更新汉化程序，都会自动把格里高尔技能里的“黎明将至”替换为“天将拂晓”


# -更新：可自定义替换内容”
下载“可自定义的程序”


首先将程序放到游戏文件的Lang目录下面


新建两个记事本old.txt和new.txt,分别输入n行字符串，可以将汉化文件中的old的第i行字符串替换成new的第i行字符串


然后生成快捷方式


Win+R打开运行栏，输入shell:startup，将快捷方式放到启动文件夹下面，重新装一遍汉化即可生效


之后不论重装还是更新汉化程序，都会自动替换
